### 2.png
- **추정 화면 타입**: Intro / Resume Summary
- **핵심 레이아웃 특징**:
  - 좌측: 프로필 사진 + 대형 헤드라인 ("더 크고 넓게 세상을 보고 싶은 이서진입니다")

  - 섹션 구분: 얇은 회색 가로선
  - 전체 배경: 화이트
- **반복 요소**: 소문자 gray 라벨, 구분선, 2열 정보 배치
- **무시한 오버레이**: 발표자 얼굴 원형(우하단), 자막 바, "Tip. 지원 회사랑 톤앤매너 맞추기" 텍스트
- **템플릿 반영 규칙**:
  - Resume는 카드 없이 순수 타이포+선으로 구성
  - label(WORK, EDUCATION 등)은 작고 gray로, 내용은 검정으로 대비

## 핵심 디자인 규칙 요약

1. **여백 우선** — 모든 화면에서 여백이 전체의 40~70% 차지
2. **타이포 위계** — 대형 제목 > gray 메타 > 본문 > caption의 4단계
3. **Red accent** — 포인트 컬러는 red 하나만, 특정 단어 또는 기여도 막대에만 사용
4. **Outline badge** — keyword는 원형 outline (fill 없음, thin stroke)
5. **기여도 막대** — 얇고 담백, red bar, 항목명 왼쪽 정렬
6. **플로팅 카드** — SNS/댓글/블로그 캡처는 white bg + soft shadow floating card
7. **목업 기울기** — 폰 목업은 rotate(-10deg) 계열 기울기 + soft shadow
8. **Thinking 섹션** — 프로젝트마다 italic/quote 형태의 회고/통찰 블록
9. **섹션 번호** — 배경에 크고 연하게 (opacity 0.08~0.12)
